    1. IPU精度问题Debug SOP¶

    当模型精度不符合预期时，在 SGS_IPU_SDK/doc/Debug_SOP/IPU精度问题Debug SOP.pdf 中，给出了几种常见的精度问题的Debug思路，具体分为以下三种：

    精度问题 精度说明 目录
    ①原始框架模型与FLOAT.SIM模型精度问题 原始网络模型与浮点网络模型结果应完全一致 1.原始框架模型与FLOAT.SIM模型精度问题
    ②FLOAT.SIM模型和FIXED.SIM模型精度问题 有一定的精度差别 2.FLOAT.SIM模型和FIXED.SIM模型精度问题
    ③板端精度问题 定点网络模型与离线网络模型结果应完全一致 3.板端精度问题
